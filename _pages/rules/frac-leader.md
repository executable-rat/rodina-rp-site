    <div class="section">
        <h2>Правила Лидеров и Заместителей</h2>
        
        <div class="subsection">
            <h3>1. Общие положения</h3>
            <ul>
                <li>1.1. Занявший пост лидера или заместителя обязуется соблюдать настоящее Положение, правила сервера и проекта.</li>
                <li>1.2. Администрация оставляет за собой право вносить изменения в Положение для устранения неясностей и злоупотреблений. Все существенные изменения анонсируются за 24 часа до вступления в силу.</li>
                <li>1.3. Санкции за нарушения применяются на основании настоящего Положения и должны быть соразмерны нарушению.</li>
                <li>1.4. Лидер несет ответственность за обучение и контроль своего состава. Систематические нарушения состава, о которых лидер знал или должен был знать, влекут санкции.</li>
                <li>1.5. Лидер вправе принимать, повышать и увольнять сотрудников на основе публичных и объективных критериев, установленных им во внутреннем уставе фракции. Увольнение без веской, задокументированной причины запрещено.</li>
                <li>1.6. От руководящего состава требуется адекватное и уважительное поведение как в игре, так и на официальных ресурсах проекта (Discord, конференции).</li>
                <li>1.7. Новый лидер назначается по итогам собеседования с Куратором фракции и Главным администратором. Действующий лидер имеет приоритетное право рекомендации преемника из числа своих заместителей.</li>
                <li>1.8. Лидер обязан заполнять и поддерживать в порядке раздел своей фракции в Discord в течение 72 часов с момента назначения.</li>
                <li>1.9. Администрация сервера оставляет за собой право принимать окончательные решения в ситуациях, неоднозначно трактуемых правилами, но явно наносящих ущерб игровому процессу, сообществу или репутации проекта.</li>
            </ul>
        </div>

        <div class="subsection">
            <h3>2. Требования к кандидатам</h3>
            <ul>
                <li>2.1. Рекомендуемый минимальный возраст — 16 лет. Возможны исключения по результатам собеседования.</li>
                <li>2.2. Кандидат должен грамотно и понятно излагать мысли в тексте и голосовом чате.</li>
                <li>2.3. Лидер должен поддерживать стабильное присутствие в игре, достаточное для управления (рекомендуемый минимум — 15 часов в неделю). Плановая неактивность более 3 дней согласуется с Куратором.</li>
                <li>2.4. Для назначения на пост лидера государственной фракции необходим игровой опыт в игре от 100 часов.</li>
                <li>2.5. Назначение заместителя согласуется с Куратором. Минимальный срок работы заместителя — один календарный месяц.</li>
                <li>2.6. Лидер обязан контролировать счёт организации и не допускать его полного истощения.</li>
            </ul>
        </div>

        <div class="subsection">
            <h3>3. Права и обязанности лидера</h3>
            <ul>
                <li>3.1. Лидер обязан знать и соблюдать правила сервера, подавая пример составу.</li>
                <li>3.2. Лидер управляет фракцией в рамках её игровой роли и утверждённого устава.</li>
                <li>3.3. Лидер поддерживает порядок, актуальность информации и активность в Discord-разделе фракции.</li>
                <li>3.4. Лидер обязан развивать роль фракции через игровые мероприятия: учения, расследования, патрули, брифинги.</li>
                <li>3.5. Лидер отвечает за здоровую атмосферу и дисциплину внутри фракции, включая предотвращение и пресечение токсичного общения.</li>
                <li>3.6. Лидер обязан разработать и опубликовать внутренние критерии для повышений (онлайн, активность, отчёты). Повышения «в день приёма» запрещены.</li>
                <li>3.7. Лидер обязан поддерживать баланс между служебной деятельностью и качественным отыгрышем роли.</li>
                <li>3.8. Лидер ведёт базовую отчётность по ключевым событиям (кадры, бюджет, ивенты) для прозрачности.</li>
                <li>3.9. Лидер вправе использовать личные игровые средства для поощрения отличившихся сотрудников и развития фракции.</li>
            </ul>
        </div>

        <div class="subsection">
            <h3>4. Запреты и ограничения</h3>
            <ul>
                <li>4.1. Запрещены: мультиаккаунтинг, обман администрации, вымогательство, торговля должностями/ценностями за реальные деньги, расизм, экстремизм.</li>
                <li>4.2. Категорически запрещена передача, продажа или предоставление доступа к своему игровому аккаунту третьим лицам.</li>
                <li>4.3. Запрещено устанавливать внутри фракции правила, противоречащие серверным.</li>
                <li>4.4. Запрещено хамское, унизительное поведение и целенаправленный троллинг.</li>
                <li>4.5. Запрещены публичные действия, направленные на подрыв авторитета Администрации проекта или дискредитацию сервера.</li>
                <li>4.6. Взаимодействие и конфликты (СВО-РП):
                    <ul>
                        <li>Разрешено: RP-конкуренция, расследования, оперативная работа, спецоперации с чётким RP-обоснованием.</li>
                        <li>Запрещено: Действия, основанные на личной неприязни, мести или предвзятом отношении, целью которых является систематическое создание препятствий для нормальной игровой деятельности другой фракции.</li>
                        <li>Масштабные акции против другой фракции требуют IC-обоснования и OOC-уведомления Кураторам минимум за 12 часов.</li>
                    </ul>
                </li>
                <li>4.7. Взаимодействие с криминалом допускается строго в рамках отыгрыша спецоперации под прикрытием или работы с информатором, с предварительным одобрением Куратора.</li>
                <li>4.8. Запрещено игнорировать официальные запросы игроков на увольнение из фракции.</li>
            </ul>
        </div>

        <div class="subsection">
            <h3>5. Арбитраж и санкции</h3>
            <ul>
                <li>5.1. Межфракционные или внутрифракционные споры, не разрешённые игровыми средствами, передаются на арбитраж Кураторов и Главной администрации.</li>
                <li>5.2. Решение арбитража, вынесенное после рассмотрения всех доказательств, является обязательным.</li>
                <li>5.3. У игроков есть право подать апелляцию на решение Куратора Главному администратору.</li>
                <li>5.4. Шкала санкций:
                    <ul>
                        <li>Выговор / Замечание: Незначительные нарушения, мелкие ошибки в отчётности.</li>
                        <li>Штраф из бюджета фракции: Нарушения, нанёсшие материальный ущерб.</li>
                        <li>Временное отстранение от должности (1-7 дней): Грубые нарушения, потеря контроля над составом.</li>
                        <li>Снятие с должности: Систематические нарушения, злоупотребление полномочиями.</li>
                        <li>Блокировка аккаунта: Мошенничество, угрозы, действия, подрывающие безопасность сервера.</li>
                    </ul>
                </li>
            </ul>
        </div>

<script>
document.addEventListener('DOMContentLoaded', function() {
    const container = document.getElementById('hierarchy-container');
    const canvas = document.getElementById('hierarchy-canvas');
    const ctx = canvas.getContext('2d');
    
    function resizeCanvas() {
        canvas.width = container.clientWidth;
        canvas.height = container.clientHeight;
        drawArrows();
    }
    
    function getEdgePoint(elementId, fromEdge, offsetPercent = 0.5) {
        const el = document.getElementById(elementId);
        if (!el) return {x: 0, y: 0};
        
        const rect = el.getBoundingClientRect();
        const containerRect = container.getBoundingClientRect();
        
        let x, y;
        
        if (fromEdge === 'top') {
            x = rect.left + (rect.width * offsetPercent) - containerRect.left;
            y = rect.top - containerRect.top;
        } else if (fromEdge === 'bottom') {
            x = rect.left + (rect.width * offsetPercent) - containerRect.left;
            y = rect.top + rect.height - containerRect.top;
        } else if (fromEdge === 'right') {
            x = rect.left + rect.width - containerRect.left;
            y = rect.top + (rect.height * offsetPercent) - containerRect.top;
        } else if (fromEdge === 'left') {
            x = rect.left - containerRect.left;
            y = rect.top + (rect.height * offsetPercent) - containerRect.top;
        } else {
            x = rect.left + rect.width / 2 - containerRect.left;
            y = rect.top + rect.height / 2 - containerRect.top;
        }
        
        return {x, y};
    }
    
    function drawArrows() {
        ctx.clearRect(0, 0, canvas.width, canvas.height);
        
        const prosecutorRight = getEdgePoint('prosecutor', 'right', 0.5);
        const governorLeft = getEdgePoint('governor', 'left', 0.5);
        
        const mvdTop = getEdgePoint('mvd', 'top', 0.5);
        const rosgvardiaTop = getEdgePoint('rosgvardia', 'top', 0.5);
        const fsbTop = getEdgePoint('fsb', 'top', 0.5);
        const minzdravTop = getEdgePoint('minzdrav', 'top', 0.5);
        
        const prosecutorBottomLeft = getEdgePoint('prosecutor', 'bottom', 0.25);
        const prosecutorBottomCenter = getEdgePoint('prosecutor', 'bottom', 0.5);
        const prosecutorBottomRight = getEdgePoint('prosecutor', 'bottom', 0.75);
        
        const governorBottomCenter = getEdgePoint('governor', 'bottom', 0.5);
        
        ctx.strokeStyle = '#8b0000';
        ctx.lineWidth = 2;
        ctx.fillStyle = '#8b0000';
        
        drawTwoWayArrow(prosecutorRight, governorLeft);
        
        drawArrow(mvdTop, prosecutorBottomLeft);
        drawArrow(rosgvardiaTop, prosecutorBottomCenter);
        drawArrow(fsbTop, prosecutorBottomRight);
        
        drawArrow(minzdravTop, governorBottomCenter);
    }
    
    function drawArrow(from, to) {
        const startX = from.x;
        const startY = from.y;
        const endX = to.x;
        const endY = to.y;
        
        ctx.beginPath();
        ctx.moveTo(startX, startY);
        ctx.lineTo(endX, endY);
        ctx.stroke();
        
        const angle = Math.atan2(endY - startY, endX - startX);
        const arrowLength = 10;
        
        ctx.beginPath();
        ctx.moveTo(endX, endY);
        ctx.lineTo(
            endX - arrowLength * Math.cos(angle - Math.PI / 6),
            endY - arrowLength * Math.sin(angle - Math.PI / 6)
        );
        ctx.lineTo(
            endX - arrowLength * Math.cos(angle + Math.PI / 6),
            endY - arrowLength * Math.sin(angle + Math.PI / 6)
        );
        ctx.closePath();
        ctx.fill();
    }
    
    function drawTwoWayArrow(point1, point2) {
        const startX = point1.x;
        const startY = point1.y;
        const endX = point2.x;
        const endY = point2.y;
        
        ctx.beginPath();
        ctx.moveTo(startX, startY);
        ctx.lineTo(endX, endY);
        ctx.stroke();
        
        let angle = Math.atan2(endY - startY, endX - startX);
        drawArrowHead(endX, endY, angle);
        
        angle = Math.atan2(startY - endY, startX - endX);
        drawArrowHead(startX, startY, angle);
    }
    
    function drawArrowHead(x, y, angle) {
        const arrowLength = 10;
        
        ctx.beginPath();
        ctx.moveTo(x, y);
        ctx.lineTo(
            x - arrowLength * Math.cos(angle - Math.PI / 6),
            y - arrowLength * Math.sin(angle - Math.PI / 6)
        );
        ctx.lineTo(
            x - arrowLength * Math.cos(angle + Math.PI / 6),
            y - arrowLength * Math.sin(angle + Math.PI / 6)
        );
        ctx.closePath();
        ctx.fill();
    }
    
    resizeCanvas();
    window.addEventListener('resize', resizeCanvas);
});
</script>
